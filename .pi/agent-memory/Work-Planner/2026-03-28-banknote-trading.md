**Project:** Banknote Trading - Static frontend website for a banknote seller.

**Tech Stack:**
- Next.js (static export)
- TailwindCSS for styling
- No backend, no database, no authentication

**Pages:**
1. Homepage - Hero segment with description
2. Browse - Grid display of all banknotes for sale
3. About Us - Seller information and qualifications
4. eBay - Explanation of eBay selling process with store link

**Workflow:** Tests first, then implementation, then QA review for each task step.

**Why:** Client needs a simple static site to showcase banknotes and direct buyers to eBay store.

**How to apply:** Prioritize static content delivery, responsive design, and clear navigation flow from Homepage → Browse → eBay purchase.
